
To compile ISSView:

OS X
	- Install the Apple development tools (from the OS X install DVD).
Windows
	- Install Visual Studio Express (untested -- please provide comments if you run into trouble).
Linux
	- Install g++ (untested -- please provide comments if you run into trouble).

OS X / Windows / Linux

- Install the Qt library (from qt.nokia.com).
- Generate a makefile for your OS (run 'qmake').
- Compile (run 'make' on OS X / Linux and 'nmake' on Windows).
